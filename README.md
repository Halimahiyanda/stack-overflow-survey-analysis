# stack-overflow-survey-analysis
Analysis of Stack Overflow Developer Survey data using data visualization to identify technology trends and workforce insights.

## Overview
This project analyzes the Stack Overflow Developer Survey dataset to identify key trends in programming languages, databases, cloud platform and developer demographics.

## Objectives
- Analyze current vs future technology trends
- Identify high-demand programming languages and databases
- Examine developer demographics
- Provide data-driven insights for decision-making

## Dataset
- Source: Stack Overflow Developer Survey
- Data includes: programming languages, databases, platforms, age, education and compensation

## Tools & Technologies
- Looker Studio (data visualization)
- PowerPoint (reporting)

- ## Key Insights
- The largest age group is **25–34 (41.3%)**, indicating a young workforce
- C# is the most desired programming language for next year
- Python and JavaScript remain widely used and in high demand
- PostgreSQL is the dominant database across current and future trends
- AWS and Microsoft Azure are the leading cloud platforms

## Visualizations
Includes:
- Programming language trends (current vs future)
- Database trends (used vs desired)
- Technology dashboards (current and future)
- Demographic analysis

## Conclusion
The analysis shows a strong shift toward modern, open-source technologies and cloud-based platforms, driven by a young and highly educated workforce.
